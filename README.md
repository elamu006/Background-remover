# 🎨 Background Remover

A simple and user-friendly web application that removes image backgrounds directly in the browser. Upload an image, process it instantly, and download the result with a transparent background.

## 🚀 Live Demo

🌐 https://elamu006.github.io/Background-remover/

## ✨ Features

- Upload images easily
- Remove image backgrounds automatically
- Preview original and processed images
- Download transparent PNG output
- Responsive design for desktop and mobile
- Fast and easy-to-use interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Background Removal API / AI Model
- GitHub Pages (Deployment)

## 📸 How to Use

1. Open the application.
2. Click **Upload Image**.
3. Select an image from your device.
4. Wait for the background removal process to complete.
5. Preview the result.
6. Download the processed image.

## 🔧 Installation

Clone the repository:

```bash
git clone https://github.com/elamu006/Background-remover.git
```

Navigate to the project folder:

```bash
cd Background-remover
```

Open `index.html` in your browser.
